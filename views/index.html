Macroni is an online music making tool built with the web audio API, socket.io, and React. Users can choose from three different instruments and connect to 'rooms' to jam with their friends.

Who/What/Where

Who/What/Where is a social app that lets users share what they're doing in the moment with people who are looking for things to do in the area. Made with a group for General Assembly's WDI program, the project uses React and the Google Maps API on the front end with Node, Express, and MongoDB on the back end.

Kastanza

Kastanza is a web application that lets users find and rate public bathrooms near them. Designed to be lightweight and mobile-friendly, this application uses jQuery with the Google Maps and Google Places APIs on the front end with a Sinatra server and PSQL on the back end.

Blocktopus

Blocktopus is a standalone game built entirely with HTML, CSS, and JavaScript. The game makes use of the canvas HTML element to manually animate all of its moving parts and is inspired by fractal trees.
